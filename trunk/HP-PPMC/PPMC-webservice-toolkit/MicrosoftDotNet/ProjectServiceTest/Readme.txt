This document describes how to use the sample .NET consumer project to demonstrate and test the
Project Service webservice for HP Project and Portfolio Management Center 7.1.

-------------------
System Requirements
-------------------
Microsoft .NET framework 2.0 or later
Microsoft Visual Studio 2005 or later
HP Project and Portfolio Management Center 7.1 running on an available and accessible computer with webservices enabled

---------------------
How to Use The Sample
---------------------
Just run the project and watch the progress of the application as it performs the various operations.

-------------------
How to Make Changes
-------------------
If you want to make changes to the application to add your own code, make modifications to the
"Program.cs" file as necessary.

-----
Notes
-----
This is a console application written using C#. There is no input required from the user except for
the URL in the beginning and a keystroke to confirm exiting when done.
All output will be visible on the console - there is no GUI.
